# IEEE-754 Converter (Flask)

## 🚀 Features
- Converts decimal numbers to IEEE-754 format
- Shows:
  - Binary representation
  - Hex value
  - Sign, Exponent, Mantissa
- Step-by-step explanation of conversion

## 🛠 Tech Stack
- Python
- Flask
- HTML/CSS

## ▶️ Run Locally
```bash
pip install flask
python app.py
